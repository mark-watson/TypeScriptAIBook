# Symbolic AI Examples

Frame-based knowledge representation and constraint programming with MiniZinc.

## Prerequisites

For MiniZinc examples: `brew install minizinc`

## Architecture

![Frame-based knowledge system with hierarchical data structures and MiniZinc constraint solver for map coloring](FIG_symbolic-AI.jpg)

## Setup

```bash
npm install
```

## Run

```bash
npx tsx frame.ts
npx tsx test_mzn.ts       # requires MiniZinc
npx tsx us_states.ts      # requires MiniZinc
```
